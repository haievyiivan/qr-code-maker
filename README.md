# qr-code-maker

A lightweight Python CLI tool to generate QR codes from URLs or text.

## Prerequisites

* Python 3.x
* `qrcode` library
* `pillow` (usually installed automatically with qrcode)

## Installation

1.  Clone the repository or download the script.
2.  Install the required dependencies:

```bash
pip install -r requirements.txt
````

## Usage

You can run the script directly from your terminal.

### 1\. Basic Usage

Generate a QR code for a URL. The filename will be auto-generated (e.g., `google_com.png`).

```bash
python main.py "https://github.com/haievyiivan"
```

### 2\. Custom Filename

Specify a specific name for your file using the `-n` flag.

```bash
python main.py "https://github.com/haievyiivan" -n "my_github_profile.png"
```

### 3\. Custom Output Directory

Save the image to a specific folder using the `-o` flag. (The script will create the folder if it doesn't exist).

```bash
python main.py "https://github.com/haievyiivan" -o "./qr_codes"
```

### 4\. Default Behavior

If you run the script without arguments, it will generate a demo QR code for the default fallback URL.

```bash
python main.py
```

## Command Line Arguments

| Argument | Flag | Description | Default |
| :--- | :--- | :--- | :--- |
| **URL** | `[text]` | The URL or text to encode. | `https://haievyiivan.github.io` |
| **Output** | `-o` / `--output` | Directory to save the image. | `.` (Current Directory) |
| **Name** | `-n` / `--name` | Custom filename (must end in .png). | Auto-generated from URL |
